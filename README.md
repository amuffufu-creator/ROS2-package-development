# AGV Base Controller

Autonomous Mobile Robot (AMR) software stack for a differential-drive robot
built on ESP32 hardware, running ROS 2 Jazzy Jalisco on Ubuntu 24.04 with
Gazebo Harmonic for simulation.

## Project status

🚧 In active development. See the project charter and schedule for milestones.

## Hardware

- Differential-drive chassis, wheel base 0.285 m, wheel radius 0.0319 m
- ESP32 microcontroller (motor control + odometry + LiDAR TCP bridge)
- RPLIDAR A1M8 laser scanner
- Encoder resolution: 330 ticks per wheel revolution

## Software stack

- ROS 2 Jazzy Jalisco on Ubuntu 24.04
- Gazebo Harmonic (simulation)
- slam_toolbox (mapping)
- Nav2 with AMCL (localization + autonomous navigation)
